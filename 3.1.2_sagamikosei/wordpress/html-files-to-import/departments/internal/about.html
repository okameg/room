内科について

ごあいさつ

当科は、患者さまのかかりつけ医としての機能と中規模病院としての専門的な機能を有し、急性期から慢性期まで一貫した治療に努めております。

消化器領域

消化器領域は早期胃癌、大腸ポリープ、消化管出血などの内視鏡治療や消化器癌に対する化学療法、C型慢性肝炎、肝硬変に対する抗ウイルス療法（インターフェロンetc.）などを中心に行い、炎症性腸疾患や一般的な消化器疾患に対しても幅広く対応しております。また随時、外科とタイアップしトータル的なケアを心がけています。

循環器領域

循環器領域は高血圧、心不全、不整脈、虚血性心疾患など、心臓や血管系に関する治療を行っております。

神経内科領域

神経内科領域は、現在入院治療は受け付けておりませんが、神経難病（パーキンソン病、脊髄小脳変性症など）や、頭痛、脳卒中などの外来診療を行っております。入院治療や、より高度の検査や治療が必要な際には、近隣の専門病院をご紹介いたします。

代謝・内分泌領域

代謝・内分泌領域は糖尿病、痛風、骨粗しょう症といった代謝疾患をはじめ、下垂体、甲状腺、副甲状腺、副腎疾患の診療を行っております。
